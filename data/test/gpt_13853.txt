Windows Server 2012 is the fifth version of the Windows Server operating system, succeeding Windows Server 2008. Windows Server 2012 was released to manufacturing on July 29, 2011, and became generally available on October 8, 2011.  Windows Server 2012 is a server operating system released by Microsoft as part of the Windows NT family of operating systems. It is the successor to Windows Server 2008, and was announced at the Microsoft Worldwide Partner Conference in May 2010.  Windows Server 2012 introduces a new architecture that allows for greater scalability and improved performance. The operating system also includes a number of new features, such as Active Directory Domain Services 7.0, which includes support for federated authentication and directory-level permissions; Windows Azure Active Directory Services; and Scale-Out File Server 2012 R2.  Windows Server 2012 is supported by a wide range of hardware, from single-processor systems to high-end servers with many processors.
