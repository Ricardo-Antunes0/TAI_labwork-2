Hermes Bulletin Board Software (BBS) by Willam Shirer  Hermes is a bulletin board system (BBS) that was released in 1984. It ran on the CP/M operating system and had a user base of mainly computer hobbyists and system administrators. Hermes was one of the first BBSes to allow users to create their own channels, which allowed for more personalized communication. The software was discontinued in 1993.
