Windows Internet Explorer 8 (IE8) is a web browser released by Microsoft as part of the Windows 8 and Windows RT operating systems. IE8 is the first major update to IE since its initial release in 2001.  IE8 introduces a new rendering engine, Trident, which provides improved performance and support for modern web standards. IE8 also includes features to help users stay safe online, including a new phishing protection feature and a password manager.
