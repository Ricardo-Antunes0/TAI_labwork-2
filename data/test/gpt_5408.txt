The Gemil Box is a multi-format box set of Japanese visual kei rock bands, released on December 5, 2009. The box set contains a CD and DVD each of three bands: B'z, The Gazette and The GazettE.  The Gemil Box is a multi-format box set of Japanese visual kei rock bands, released on December 5, 2009. The box set contains a CD and DVD each of three bands: B'z, The Gazette and The GazettE.   The Gemil Box is named after the founder and CEO of Gemil Records, Yasushi Inoue.
