TRON Code is a multi-byte character encoding scheme for the Chinese language. It is defined in TRON Project Working Document TR-CSCW-06, and was created by the TRON Foundation in June 2006.  TRON Code is a multi-byte character encoding scheme for the Chinese language. It was created by the TRON Foundation in 2006. It is based on the GB2312 encoding scheme, but uses a different byte order and has additional symbols.
