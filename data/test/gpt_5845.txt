Log4Shell (CVE-2021-44228) was a zero-day vulnerability in the Log4j logging library that could be used to execute arbitrary code on targeted systems.  Log4Shell was discovered by Mandriva Security Team on February 2, 2021, and was publicly disclosed on February 6, 2021.   Log4Shell is a zero-day vulnerability in the Log4j logging library. It could be used to execute arbitrary code on targeted systems.
