KStars is a freely licensed planetarium program for Linux and other Unix-like systems. It is based on the KDE Frameworks and uses the Qt library.  KStars is a free software project that aims to provide a comprehensive planetarium program for Linux and other Unix-like systems. It is based on the KDE Frameworks and uses the Qt library. KStars has been in development since 2006 and has seen several updates, including a new user interface and support for 3D viewing.
